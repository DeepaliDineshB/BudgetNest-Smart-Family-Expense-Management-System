# BudgetNest - Smart Family Expense Management System

💰 BudgetNest - Smart Family Expense Management System

A Django-based web application that helps families efficiently manage their income, expenses, savings, and monthly budgets. The application provides secure user authentication, expense tracking, budget management, CRUD operations, and report generation using Django and MySQL.

---

## 🚀 Features

- Secure User Authentication (Login & Registration)
- Add, Edit, Delete, and View Expenses
- Income Management
- Monthly Budget Planning
- Expense Categorization
- Real-time Expense Tracking
- Financial Reports and Summary
- Responsive User Interface
- Data stored securely using MySQL

## 🧠 Application Workflow

1. User registers or logs into the system.
2. User adds monthly income.
3. User records daily expenses.
4. Expenses are categorized automatically.
5. Budget usage is calculated in real time.
6. Users can update or delete records anytime.
7. Dashboard displays financial summary and reports.
